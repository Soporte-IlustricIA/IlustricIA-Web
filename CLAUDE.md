# Proyecto: IlustricIA — Agencia de Inteligencia Artificial

## Stack técnico
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- react-router-dom (rutas: `/`, `/demo`, `/privacy`, `/legal`, `/cookies`)
- framer-motion para animaciones
- Dominio: ilustricia.com

## Contexto SEO del negocio

DATOS BÁSICOS:
Nombre: IlustricIA
Web: ilustricia.com
Descripción: Agencia especializada en integración de soluciones prácticas de IA
para empresas. Metodología: diagnóstico → roadmap por fases → implementación → métricas.

SERVICIOS:
- Chatbots para web, WhatsApp y otros canales con flujos personalizados
- Asistentes internos para equipos (ventas, soporte, operaciones)
- Automatización de ventas: captación de leads, scoring automático,
  follow-ups inteligentes por email/WhatsApp
- Desarrollo web profesional orientado a conversión
- Marketplaces a medida (experiencia completa: registro → pago)
- Atención al cliente con IA (reducción de tiempos de respuesta)

CLIENTE IDEAL:
Pymes y empresas en crecimiento que quieren aplicar IA de forma práctica.
Sectores destacados: restauración, hostelería, retail, servicios profesionales.
No requieren datos perfectos para empezar — IlustricIA evalúa qué existe y qué falta.

PROPUESTA DE VALOR DIFERENCIAL:
- Implementación el mismo día (sin dependencia de proveedor externo)
- Acompañamiento cercano durante todo el proceso
- Resultados medibles con KPIs definidos desde el inicio
- Primer piloto/MVP listo en pocas semanas
- Comercialización B2B de soluciones IA con comisión sobre proyectos

CÓMO TRABAJAR EN ESTE PROYECTO:

- Comparativas siempre en tabla
- Output implementation-ready: titles exactos, H1s, metas, slugs, anchors
- El operador implementa el mismo día — no dar instrucciones vagas
- No inventar datos de volumen de búsqueda sin acceso a herramientas
